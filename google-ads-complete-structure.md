# Estrutura Completa: Google Ads High Performance (Nova Jaguari)

Este documento contém todos os ativos necessários para criar uma campanha de alta conversão. **Copie e cole** os textos conforme indicado no painel do Google Ads.

---

## ⚙️ Configurações da Campanha (Settings)

- **Objetivo:** Criar uma campanha sem meta (ou Leads).
- **Tipo:** Pesquisa (Search).
- **Redes:** DESATIVAR "Rede de Display". Manter apenas "Rede de Pesquisa".
- **Locais:** 
    - Santana de Parnaíba (Cidade)
    - Barueri / Alphaville (Raio de 15km)
    - Opcional: São Paulo (Bairros: Itaim Bibi, Vila Olímpia, Moema, Pinheiros).
- **Idiomas:** Português e Inglês (importante para quem usa SO em inglês).
- **Lances (Bidding):** Maximizar Conversões.
- **Orçamento Diário:** R$ 133,00 (Totalizando ~R$ 4.000,00/mês).
